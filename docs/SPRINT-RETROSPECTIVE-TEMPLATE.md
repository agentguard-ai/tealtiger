## Template Usage Instructions

1. **Schedule retrospective** at the end of each sprint (1 hour)
2. **Gather metrics** before the meeting
3. **Use this template** to guide the discussion
4. **Focus on actionable items** rather than blame
5. **Save completed retrospective** in `docs/sprints/sprint-[number]-retrospective.md`
6. **Track action items** in next sprint planning
