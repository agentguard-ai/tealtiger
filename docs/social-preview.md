# Social Preview Image

The repo uses `.github/social-preview.png` as the GitHub social preview image.

To set or update it:

1. Open the repository on GitHub.
2. Go to `Settings`.
3. Open `General`.
4. Scroll to `Social preview`.
5. Upload `.github/social-preview.png`.
6. Save the change.

The current image is sized at `1280x640` so it stays readable in link cards and
social embeds.
