## üí° Tips for Dev.to Image Generator

### If the first result isn't perfect:

1. **Try regenerating** - AI generators give different results each time
2. **Adjust the prompt** - Add or remove details
3. **Focus on key elements** - Simplify if too complex
4. **Emphasize text** - Add "large bold text" or "clear readable text"

### Key phrases to include:
- "professional"
- "high contrast"
- "modern"
- "tech aesthetic"
- "clean design"
- "bold text"
- "16:9 aspect ratio"

### Key phrases to avoid:
- "realistic photo" (illustrations work better)
- "people" or "faces" (can look weird in AI)
- Too many specific details (AI gets confused)
