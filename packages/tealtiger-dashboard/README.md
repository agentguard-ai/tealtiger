# TealTiger Dashboard

Core React dashboard shell for TealTiger governance views.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run build
npm test
npm run build-storybook
```

## Layout Scope

- responsive app shell with desktop sidebar and mobile navigation drawer
- dark mode default with light mode toggle
- routed sections for Overview, Agents, Events, Cost, Security, Compliance, and Settings
- header search, notifications control, agent status indicators, and breadcrumbs
- Storybook stories for shell, sidebar, header, breadcrumbs, theme control, and placeholder content
