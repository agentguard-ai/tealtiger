# Welcome to TealTiger

TealTiger is a **developer-first AI security and cost governance SDK** that provides guardrails, policy enforcement, and audit logging for LLM applications. Built for developers who need production-ready security without infrastructure complexity.

## Why TealTiger?

<CardGroup cols={2}>
  <Card title="Zero Infrastructure" icon="server">
    SDK-only architecture. No servers, no deployment complexity. Integrate directly into your application code.
  </Card>
  <Card title="Policy-as-Code" icon="code">
    Define policies in code with rollout modes (ENFORCE, MONITOR, REPORT_ONLY) for safe production deployment.
  </Card>
  <Card title="Deterministic Decisions" icon="check-circle">
    Stable, typed Decision contract with action, reason codes, and risk scores for reliable policy enforcement.
  </Card>
  <Card title="Security by Default" icon="shield-halved">

  </Card>

## Key Features

### Policy Rollout Modes
Deploy policies safely with three modes:
- **ENFORCE**: Block violating actions
- **MONITOR**: Log violations without blocking
- **REPORT_ONLY**: Collect data for policy tuning

### Audit & Compliance
- Versioned audit schema with redaction-by-default
- Correlation IDs for distributed tracing
- OWASP ASI-aligned security controls

### Developer Experience
- TypeScript and Python SDKs
- Policy testing framework with CLI
- CI/CD integration ready
- Zero telemetry to external servers
